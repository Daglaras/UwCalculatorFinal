# Profilco U-Value Calculator

## Quick Start (2 steps!)

### Prerequisites (one-time only)
1. **Python**: Download from https://www.python.org/downloads/
   - Windows: ⚠️ Check "Add Python to PATH" during installation!
2. **Node.js**: Download from https://nodejs.org/ (LTS version)

---

## Windows
1. Extract the zip folder
2. Double-click **`start-windows.bat`**
3. The app will open in your browser automatically!

## Mac
1. Extract the zip folder
2. Double-click **`start-mac.command`**
   - If you see "cannot be opened" error: Right-click → Open → Open
3. The app will open in your browser automatically!

---

## Usage

1. Select **Window Type** (Sliding/Opening)
2. Select **Series** (PR320, PR45, IQ34, IQ460, IQ580)
3. Select **Category** (Δίφυλλο, Τρίφυλλο, Τετράφυλλο)
4. Select **Driver** (case profile)
5. Select **Sash** (glass profile)
6. Enter **Dimensions** (height, width, Ug, Ψ)
7. Click **Calculate** → Get your Uw result!

---

## Stopping the App

- **Windows**: Press any key in the black terminal window
- **Mac**: Press Ctrl+C in the terminal or close the terminal window

---

## Troubleshooting

### "Python is not installed"
- Download from https://www.python.org/downloads/
- Windows: Make sure to check "Add Python to PATH"!

### "Node.js is not installed"
- Download from https://nodejs.org/

### App doesn't open in browser
- Open manually: http://localhost:3000

### Port already in use
- Close other applications using ports 3000 or 8000
- Or restart your computer
